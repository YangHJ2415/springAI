# 🤖 AI 메모리 채팅 서비스 (Spring AI + Groq LLaMA-4)  

Groq의 LLaMA 4 Scout LLM을 활용하여,  
사용자와의 대화를 기억하고 이어나갈 수 있는,  
간단한**메모리 기반 AI 채팅 예제**입니다.

---

## 🚀 기술 스택

| 항목 | 사용 기술                                  |
|------|----------------------------------------|
| 📦 백엔드 | Spring Boot                            |
| 🤖 AI 모델 | Groq의 Meta LLaMA 4 Scout               |
| 🧠 AI 연동 | Spring AI (spring-ai-chat, chat-memory) |
